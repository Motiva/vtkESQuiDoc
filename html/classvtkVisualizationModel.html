Detailed Description

vtkVisualizationModel object defines a visualization mesh (vtkPolyData)

This class inherits from vtkModel base class. As it is specified in vtkModel, at least an input mesh should be defined. Optionally a source mesh for synchronization purposes may be defined. To enhance visualization user experience a mesh texture can be specified through the TextureFileName attribute.
