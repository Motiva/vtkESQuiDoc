Detailed Description

Generic base class for collision detection library interfaces.

This class serves as a base class for the collision detection libraries. Scenario object models, vtkCollisionModel must be assigned before process execution. The result of the procedure is collection of collisions, vtkCollisionCollection, between scenario objects. vtkBioEngInterface, an interface based on this class, has been implemented to access the available deformation model, vtkbioeng.
