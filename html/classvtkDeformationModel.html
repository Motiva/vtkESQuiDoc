Detailed Description

vtkDeformationModel object defines a deformation model based on a mesh (vtkPolyData).

This class inherits from vtkModel base class. As it is specified in vtkModel, at least an input mesh should be defined. Optionally a source mesh for synchronization purposes may be defined. An interface based on this class must be implemented to access available deformation models: vtkParticleSpringSystem, vtkExplicitDeformableModel, etc...

Initialize the deformation model.

The model has to be initialized in order to be updated. At least one of these 2 parameters must be previously defined:

  • Input: vtkPolyData object
  • FileName: path to a .vtp file.
