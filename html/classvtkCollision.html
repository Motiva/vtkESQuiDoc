Detailed Description

This class contains the required information to define a collision.

vtkCollision acts as data container storing all the useful information of a scenario collision:

  • Type
  • ObjectId
  • ModelId
  • Cell Identifier
  • Cell Normal
  • Point Identifier
  • Point Coordinates
