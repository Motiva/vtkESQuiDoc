Detailed Description

Implementation class for scenario organ definition.

In vtkESQui an organ is considered an scenario object, so vtkScenarioObject serves as its base class.
As a vtkModel based class, at least a visualization model has to be defined. If the organ is deformable, a deformation and a collision model must also be defined.
