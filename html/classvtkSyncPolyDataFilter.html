Detailed Description

This class represents an abstract model on a scenario element.

It acts as a base class for visualization, collision and deformation models. vtkSyncPolyDataFilter class has 2 inputs. By default the input data will be read from a vtp file specified in FileName attribute. A second optional parameter is available, the source mesh. Main input will be adapted to the source mesh (point positions will be synchronized) The output displays the transformed mesh according to transformation matrix values.
