Detailed Description

vtkCollisionModel object defines a collision model based on a mesh (vtkPolyData).

This class inherits from vtkModel base class. As it is specified in vtkModel, at least an input mesh should be defined. Optionally a source mesh for synchronization purposes may be defined. Collision model must be synchronized with deformation model if present.
